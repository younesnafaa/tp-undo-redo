Exercice "Polyline"

Enoncé
Implémenter avec la bibliothèque xState un statechart qui trace une "polyline" (suite de points interconnectés)
  • MOUSECLICK ajoute un point (jusqu'au maximum MAX fixé)
  • MOUSEMOVE déplace le point provisoire
  • Backspace efface le dernier point ajouté (sauf le premier)
  • Enter enregistre la polyline (2 ≤ nombre de points ≤ MAX)
  • Escape abandonne le tracé de la polyline
Vous pouvez utiliser l'éditeur xState en ligne et copier / coller le code généré dans votre projet ou installer l'extension xState pour VSCode pour éditer le statechart directement dans VSCode.
